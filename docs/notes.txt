**CHECKPOINT RESUME FUNCTIONALITY (2025-07-08):**
✅ **Added optional training resume capability**

**NEW FEATURE:**
- Added `--resume` flag to training command
- Default: starts fresh training (current behavior unchanged)
- With `--resume`: automatically finds and loads latest checkpoint
- Shows checkpoint info (epoch, global_step) when resuming

**USAGE:**
- Fresh training: `./env/bin/python -m src.main --mode train`
- Resume training: `./env/bin/python -m src.main --mode train --resume`

**BENEFITS:**
- Continue training after interruptions (power loss, manual stop)
- Experiment with different epochs without losing progress
- Preserve training state (optimizer, learning rate scheduler)
- Automatic checkpoint detection (finds latest by modification time)

**SAFETY:**
- Falls back to fresh training if no checkpoints found
- Validates checkpoint compatibility before loading
- Clear logging of resume vs fresh training status

**READY TO TRAIN:** Run `./env/bin/python -m src.main --mode train`
