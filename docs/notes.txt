**HYPERPARAMETER OPTIMIZATION (2025-07-08):**
✅ **Optimized for Apple M1 Pro + 16GB + 5 years crypto data**

**KEY OPTIMIZATIONS MADE:**
- TFT_HIDDEN_SIZE: 32 → 128 (4x increase for crypto complexity)
- TFT_HIDDEN_CONTINUOUS_SIZE: 8 → 32 (4x increase for richer features)
- TFT_ATTENTION_HEADS: 4 → 2 (reduced for crypto short-term patterns)
- TFT_DROPOUT: 0.2 → 0.3 (increased for crypto volatility/noise)
- TFT_LEARNING_RATE: 1e-3 → 3e-4 (reduced for stability with larger model)
- TRAINING_EPOCHS: 10 → 50 (5x increase for proper convergence)
- TFT_BATCH_SIZE: 128 → 64 (optimized for M1 Pro memory efficiency)
- TFT_GRADIENT_CLIP_VAL: 0.1 → 0.5 (increased for gradient stability)
- TFT_EARLY_STOP_PATIENCE: 5 → 8 (more patience for larger model)
- TFT_ACCELERATOR: "cpu" → "mps" (Apple Silicon GPU acceleration)
- TFT_NUM_WORKERS: 0 → 4 (utilize M1 Pro's 10 cores efficiently)

**RATIONALE (based on research):**
- Larger hidden_size for crypto complexity and 5-year dataset
- Lower attention heads for short-term crypto patterns (vs long-term forecasting)
- Higher dropout for noisy crypto market data
- Conservative learning rate for stable training
- MPS acceleration for 2-5x speedup on Apple Silicon
- Batch size optimized for 16GB memory + MPS efficiency
